Smart Transit System

Complete Technical Project Report & Documentation

1. Objective

To develop a modern, real-time bus tracking and booking application with a focus on user experience, administrative control, and efficient urban commuting.

2. Technology Stack

React.js Node.js Express MongoDB Socket.io Leaflet.js Glassmorphism UI

The system utilizes a modern full-stack architecture with real-time bi-directional communication between the server and multiple clients.

3. Key Functionalities

📍 Real-time Tracking

Buses move across an interactive map using a simulation algorithm. Users can "Track" specific buses, triggering a smooth map recenter and movement tracking.

🎫 Interactive Booking

A 2D seat grid allows users to reserve specific spots. The system handles availability in real-time to prevent double bookings.

🛡️ Admin Dashboard

A floating administrative interface allows for fleet management and user oversight without interrupting the core user experience.

4. Core Technical Implementation

Backend Simulation Logic

// server.js - Real-time movement logic
setInterval(async () => {
    const buses = await Bus.find({});
    for (let bus of buses) {
        let newLat = bus.lat + (Math.random() - 0.5) * 0.002;
        let newLng = bus.lng + (Math.random() - 0.5) * 0.002;
        await Bus.updateOne({ busId: bus.busId }, { lat: newLat, lng: newLng });
    }
    const updatedBuses = await Bus.find({});
    io.emit('bus_locations_updated', updatedBuses.map(b => ({ 
        busId: b.busId, lat: b.lat, lng: b.lng 
    })));
}, 3000);

Frontend Map Recenter Hook

// App.js - Leaflet stabilization hook
function RecenterMap({ lat, lng }) {
  const map = useMap();
  useEffect(() => { 
    map.setView([lat, lng]); 
  }, [lat, lng, map]);

  useEffect(() => {
    setTimeout(() => { map.invalidateSize(); }, 400);
  }, [map]);
  return null;
}

5. Design Philosophy

The application employs a Glassmorphic design—utilizing semi-transparent panels with backdrop-blur effects. This creates a high-end, futuristic feel while maintaining readability.

  • Typography: "Outfit" font family for a clean, geometric aesthetic.
  • Color Palette: Vibrant blue accents (#3b82f6) against deep slate backgrounds.
  • Animations: Smooth CSS transitions for all interactive elements.

6. Conclusion

The Smart Transit System successfully integrates complex real-time data with a user-centric design, providing a scalable foundation for future urban projects.
