Smart Transit

Real-Time Bus Tracking & Booking System

Minor Project • 2026 • github.com/Pumpkaboo123/bus

😤 The Problem

  • People don't know where the bus is or when it will arrive
  • Long queues just to buy a simple ticket
  • No way to reserve a seat in advance
  • Bus operators have no digital tools to manage their fleet

💡 Our Solution

A web application where you can see buses on a live map, pick your seat, and get a QR ticket — all in real time!

Think of it like Uber, but for city buses.

🛠️ Technology Stack

🖥️ React.js

Builds the user interface — buttons, pages, forms.

⚙️ Node.js

The "brain" — handles login, booking, and tracking logic.

💾 MongoDB

The "memory" — stores users, buses, and bookings.

📡 Socket.io

Live updates — sends bus positions every 3 seconds.

⚡ How It Works

  • Step 1: User opens the website & logs in
  • Step 2: They see a sidebar with bus cards + a live map
  • Step 3: Click "TRACK" → map follows the bus
  • Step 4: Click "BOOK NOW" → pick seats on a grid
  • Step 5: Confirm → get a ticket with a QR code
  • Step 6: Print the ticket or show it on your phone!

📡 Live Tracking Code

Every 3 seconds, buses move on the map:

setInterval(async () => {
    const buses = await Bus.find({});
    for (let bus of buses) {
        bus.lat += (Math.random() - 0.5) * 0.002;
        bus.lng += (Math.random() - 0.5) * 0.002;
        await bus.save();
    }
    // Tell ALL users about new positions
    io.emit('bus_locations_updated', buses);
}, 3000);  // Runs every 3 seconds

🔐 Login System Code

socket.on('login', async (data) => {
    const user = await User.findOne({
        username: data.username,
        password: data.password
    });

    if (user) {
        socket.emit('auth_success', user);  // ✅ Welcome!
    } else {
        socket.emit('auth_error', 'Wrong details'); // ❌
    }
});
